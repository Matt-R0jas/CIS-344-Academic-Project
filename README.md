# CIS-344-Academic-Project
Academic Research Collaboration Project
The Academic Research Collaboration Platform is a PHP/MySQL web application for university/researchers to collaborate by
managing projects and sharing documents securely.

NEED TO RUN:
The latest version of PHP
MySQL with arc_database (will be provided)
Apache server

Setup:
1. Place all PHP files in your web server's document root.
2. Copy the SQL code to create the 'users', 'project', and 'documents' tables.
3. You may have to update the database credentials in the db.php.

How to Run:
1. Start Apache and MySQL with XAMPP.
2. Open a browser and go to http://localhost/ARC_Project/.
3. Register as a new user.
4. Log in with your account.
5. Create a new Project from the dashboard.
6. Visit a project page to upload files.
7. Uploaded files can be downloaded from the Project page.
8. Log out when you feel like it.
